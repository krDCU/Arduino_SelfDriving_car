## 아두이노와 RC카를 이용한 자율주행 구현

+ 사용언어 : C++
+ 사용 소프트 웨어 : Arduino, SolidWorks, Ultimaker Cura
+ 제작기한 : 2022.03 ~ 2022.06 ( 4개월 소요 )
+ 제작인원 : 2명
 + 역할 : 센서 연결 및 기본적인 구동 동작 프로그래밍, SolidWorks를 이용해 센서를 고정시킬 모델링 제작 후 3D Print로 출력
